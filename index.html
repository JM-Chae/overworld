들어가며

이 웹 페이지는 본격적으로 HTML을 공부하며 작성자가 처음 만든 웹 페이지이다. 공부를 진행하며 점진적으로 웹 페이지의 개선을 실시할 것이며, 좀 더 나은 웹 페이지를 향해 정진해 나가는 것을 목표로 한다.

이 웹 페이지에는 복습을 위한 개인용 노트이자 그때 그때 떠올린 잡념들을 게시하는 일기장처럼, 지극히 작성자의 개인적인 용도로 활용할 예정이다.

이 웹 페이지는 한국어와 일본어 두 언어로 작성할 예정이며, 모든 게시글은 각 언어로 작성되어 방문자로 하여금(작성자의 지인 외에 방문하는 사람이 있을지는 모르겠지만) 언어를 선택하여 열람할 수 있게 하는 기능을 추가하고 싶다.
